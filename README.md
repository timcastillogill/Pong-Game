# Pong Game in Python

## How to play?

1. Clone the project and open in your IDE
2. Right click in the code window and click 'Run Python File in Terminal'

---
## **What are the buttons to move the paddles?**

### Left Paddle

Up = 'w'
Down = 's'

### Right Paddle

Up = Up Arrow
Down = Down Arrow
